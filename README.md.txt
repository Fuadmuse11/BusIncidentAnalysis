# Bus Incident Analysis Report

This Power BI report analyzes bus incidents from 2015 to 2024, focusing on Transport providers such as Arriva, Metroline, and others. It provides detailed insights into borough-level data, casualty categories, and incident details. The goal of this report is to identify which borough has the highest number of incidents, understand trends over time, and evaluate which transport provider is involved in the most incidents.

## Features
- **Borough-Level Analysis**: Identify which borough has the highest number of incidents and track trends over time.
- **Operator Performance**: Compare transport providers, such as Arriva and Metroline, to evaluate their involvement in incidents.
- **Incident Categories**: Analyze casualty categories and the severity of incidents.
- **Time-Based Insights**: Understand how incidents have evolved from 2015 to 2024.

## Instructions
1. Open the `BusIncidentAnalysis.pbix` file in Power BI Desktop to begin exploring the report.
2. Navigate through the report using the two buttons on the **Report** page:
   - **Borough Insights**: Takes you to the borough-level analysis page.
   - **Incident Insights**: Directs you to the page analyzing transport providers and detailed incidents.
3. Interact with the visuals, slicers, and tooltips to explore specific trends by time period, borough, or transport provider.

## File Contents
- `BusIncidentAnalysis.pbix`: Power BI report file containing all visuals and insights.
- `bus-fatalities-2024-q2.xlsx`: Dataset with details of incidents from 2015 to 2024.
